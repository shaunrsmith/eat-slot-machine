🔑 One-time setup

This app reads live restaurants from Google Places. Paste a Google Maps API key — it's stored only in this browser (never uploaded).

How do I get a key? (5 min, free)
  1. Go to Google Maps Platform and create/select a project.
  2. Enable these APIs: Maps JavaScript API, Places API (New), Geocoding API.
  3. Credentials → Create credentials → API key.
  4. Restrict it: Application restrictions → Websites, add https://shaunrsmith.github.io/* (and http://localhost/* for local testing).
  5. Paste it above. Personal use stays within Google's free monthly tier.
